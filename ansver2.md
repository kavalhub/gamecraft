1. Игрок изучает рецепт "Деревянный меч" (type: blueprint)
2. Создаётся чертёж (items.stage: blueprint, recipe_slug: craft_wooden_sword)
3. Игрок крафтит: чертёж + 5 wood → предмет (items.stage: item)
4. Предмет можно разобрать → чертёж + ресурсы

да впоминай:
**Процесс:**
1. Игрок выбирает чертёж из инвентаря
2. Загружает материалы в слоты согласно рецепту
3. (Опционально) Вводит кастомное название
4. Нажимает "Создать"
5. Чертёж трансформируется в предмет

**Автоматическое название:** конкатенация материалов + тип  
Пример: `"Осиновый железный могильный посох"`

**Кастомное название:** игрок может назвать предмет как угодно  
Пример: `"Осиное жало"` или `"Дерьмо на палке"`

чертёж содержит в формуле типы ресурсов и их количество
предмет состоит из конкретных ресурсов тип которых описан в чертеже
